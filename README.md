📊 Website Engagement & Bounce Rate Analysis (Power BI)
🔎 Overview

This project analyzes user behavior on an e-commerce platform using the Google Merchandise Store Analytics Dataset. The primary goal is to understand how bounce rate impacts user engagement and overall website performance, and to identify areas for improving user retention.

🎯 Objectives
Analyze bounce rate across different user sessions
Study engagement using metrics like pageviews and hits
Identify high bounce rate pages and traffic sources
Understand the impact of device type and user location
Suggest strategies to improve engagement and reduce bounce rate
📁 Dataset

Google Merchandise Store Analytics Dataset

Contains user session-level data
Includes attributes such as:
Page views
Traffic source
Device type
Country
Session activity (hits)
Used to analyze user interaction and engagement patterns
🛠 Tools & Technologies

Microsoft Power BI • Data Cleaning • Data Visualization • DAX (Data Analysis Expressions)

🔄 Project Workflow

Data Collection → Data Cleaning → Data Transformation → DAX Calculations → Dashboard Creation → Insights Generation

📊 Key Metrics (Calculated)
Total Sessions
Pages per Visit
Hits per Session
Bounce Rate

👉 Bounce Rate Formula:
Bounce Rate = (Sessions with only 1 Page View / Total Sessions) × 100

📈 Dashboard Analysis

Key visualizations include:

Bounce Rate by Month (Trend Analysis)
Engagement Level vs Bounce Category
Visitor Type vs Engagement & Bounce Rate
Distribution of Bounce Categories
Overall Performance KPIs (Sessions, Conversion Rate, Engagement Score)

📌 Conclusion

The analysis shows that better user interaction leads to lower bounce rates and improved website performance. By optimizing content, navigation, and user experience, businesses can significantly increase engagement and conversions.
