# 📊 Website Engagement & Bounce Rate Analysis

## 🔎 Overview

This project analyzes user behavior on an e-commerce website using the **Online Shoppers Intention Dataset**.
The goal is to understand how **bounce rate affects user engagement and purchasing behavior** and suggest ways to reduce user drop-offs.

---

## 🎯 Objectives

* Analyze bounce rate distribution across sessions
* Study user engagement through page interactions
* Identify factors influencing purchase decisions
* Suggest methods to improve engagement

---

## 📁 Dataset

* **Online Shoppers Intention Dataset**
* **12,330 user sessions**
* Contains browsing behavior, session duration, page visits, and purchase outcomes.

---

## 🛠 Tools & Technologies

Python • Pandas • NumPy • Matplotlib • Seaborn • Scikit-learn

---

## 🔄 Project Workflow

Data Collection → Data Cleaning → Exploratory Data Analysis → Feature Engineering → Logistic Regression Model → Evaluation

---

## 📈 Exploratory Analysis

Key visualizations include:

* Bounce Rate Distribution
* Product Pages Visited
* Visitor Type Distribution
* Bounce Rate vs Product Pages
* Bounce Rate vs Page Duration
* Bounce Rate vs Page Value

---

## 🤖 Machine Learning Model

A **Logistic Regression model** was used to predict purchase behavior.

**Model Accuracy:** 88%

Important features:

* PageValues
* ProductRelated_Duration
* Month
* Weekend activity

---

## 💡 Key Insights

* Most sessions show **low bounce rates**, indicating good engagement.
* **Returning visitors dominate** website traffic.
* More product page visits generally lead to **lower bounce rates**.

---

## 🚀 Recommendations

* Improve product page content and design
* Optimize website loading speed
* Enhance navigation and UI
* Add personalized recommendations

---

## 📌 Conclusion

Higher user interaction reduces bounce rate and increases purchase probability. Improving product page experience can significantly enhance engagement and conversions.
